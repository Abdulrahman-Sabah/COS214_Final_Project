Detailed Description

Abstract base class implementing the State pattern for plant lifecycle management.

This class defines the interface for all plant lifecycle states and provides common functionality for state management. Each concrete state represents a different phase in a plant's lifecycle (Seedling, Growing, Mature, etc.).

\doxysubsection{Detailed Description}
Simple concrete implementation of the \doxylink{class_plant}{Plant} interface. 

\doxylink{class_basic_plant}{Basic\+Plant} provides a straightforward implementation of the \doxylink{class_plant}{Plant} abstract class with basic properties and functionality. It can be used as a generic plant type or as a base for more specialized plant implementations.

\doxysubsection{Detailed Description}
Represents the active growing state in a plant\textquotesingle{}s lifecycle. 

\doxylink{class_growing_state}{Growing\+State} is a concrete state in the State pattern representing a plant that is actively growing. From this state, plants can either advance to dormancy or die if conditions are unfavorable.

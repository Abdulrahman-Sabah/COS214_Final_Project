\doxysubsection{Detailed Description}
Concrete builder class for constructing \doxylink{class_cactus}{Cactus} objects. 

This class implements the Builder pattern to construct \doxylink{class_cactus}{Cactus} plants step by step, allowing for flexible configuration of cactus properties.

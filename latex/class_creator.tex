\doxysubsection{Detailed Description}
Abstract base class for the Factory Method pattern. 

The \doxylink{class_creator}{Creator} class defines the interface for creating \doxylink{class_plant}{Plant} objects using the Factory Method design pattern. Concrete creators will inherit from this class and implement the factory\+Method to create specific types of plants.

\doxysubsection{Detailed Description}
Concrete command class for moving plants to sales area. 

This class implements the \doxylink{class_commands}{Commands} interface to represent a move-\/to-\/sales command in the Command pattern. It encapsulates the request to move a specific plant to the sales area, typically handled by the \doxylink{class_greenhouse_manager}{Greenhouse\+Manager}.

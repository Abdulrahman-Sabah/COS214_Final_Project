\doxysubsection{Detailed Description}
Represents the selling state in a plant\textquotesingle{}s lifecycle. 

This state represents when a plant is ready for sale. It\textquotesingle{}s the final state in the plant lifecycle where the plant is mature enough to be sold to customers.

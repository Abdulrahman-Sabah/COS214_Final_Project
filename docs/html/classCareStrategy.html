Detailed Description

Strategy interface for applying plant-specific care.

Defines the structure for care behavior such as watering, pruning, fertilizing, or seasonal actions. Concrete strategy subclasses like CactusCare, RoseCare, and LavenderCare implement their own logic for the applyCare() method.
